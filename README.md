# LandingSite
+Simple HTML landing site containing LOGIN, REGISTER forms using Bootstrap css


+ This repo contains 2 html files and an image.
+ image is used as bg only in dark-mode.html file . Dark mode toggle button is also used in this file.
+ And the other html file is just simple and does not have any bg and dark mode toggle button.


#dark-mode.html 
It contains hame, about, login form, register form and a dark mode toggle button
+ Dark mode is applied to body content only
+ Forms do appear on home page itself
